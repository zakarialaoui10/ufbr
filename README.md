# ufbr
universal file based router


## Usage 

```js
 import {createFileBasedRouter} from 'ufbr/[FRAMEWORK]'
 createFileBasedRouter({
    pages : import.meta.glob('./pages/**/*.[jsx, js]'),
    target : document.body
 })
```

## Features
- File Based Routing 
- Sync and Async Components
- Nested Routes 
- Dynamic Routes
